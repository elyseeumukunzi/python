# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added
- Initial release of Bulk Background Remover
- Offline AI background removal using rembg
- Batch processing support for multiple images
- Smart image resizing with aspect ratio preservation
- Transparent PNG output
- Progress tracking and status updates
- Comprehensive error handling
- Support for JPG, JPEG, PNG, and WEBP formats
- Windows batch file for easy execution
- Complete documentation (README, QUICKSTART, EXAMPLES)
- MIT License
- Contributing guidelines

### Features
- Maximum dimension configuration (default: 500x500px)
- Automatic output folder creation
- Detailed processing summary
- Failed file tracking and reporting
- Optimized PNG compression

### Documentation
- Professional README with badges
- Quick start guide
- Usage examples and use cases
- Contributing guidelines
- GitHub publishing guide
- MIT License
